Philosophers

Developed a robust solution to the Dining Philosophers problem in C, employing mutexes and threads for efficient synchronization. Our implementation showcased adeptness in concurrent programming, handling complex resource allocation seamlessly. Through meticulous threading and synchronization techniques, we ensured seamless resource management, illustrating our adeptness in handling intricate concurrent programming challenges.

My Role: In this project, I addressed the Dining Philosophers problem in C, utilizing mutexes to manage access to shared forks. Each philosopher was modeled as a thread, facilitating concurrent execution. Through meticulous design and optimization, I ensured a robust solution, demonstrating expertise in concurrent programming.

Push Swap

In Push Swap, I devised an innovative sorting algorithm to efficiently arrange numbers in ascending order using two stacks, namely 'a' and 'b'. Leveraging the radix sort algorithm with bit shifting techniques, I optimized the sorting process for enhanced performance and precision.

The Push Swap project challenges the sorting of numbers in ascending order, employing two stacks, 'a' and 'b', with precise movements. By integrating the radix sort algorithm with bit shifting techniques, this endeavor optimizes sorting efficiency to its fullest potential. This project highlights the fusion of algorithmic ingenuity and efficient data structure management, culminating in optimal sorting solutions.

C Algorithms Radix Sort Bit Shifting Data Structures

Fract-ol

Fract-ol project delves into the mesmerizing world of fractals, leveraging the MLX library in C to create captivating visualizations of intricate self-similar patterns inspired by mathematical abstraction and natural phenomena.

Created Mandelbrot, Julia, and Tricorn fractals in C using MLX library. Implemented algorithms to calculate escape iterations for each fractal type, mapping coordinates and assigning colors accordingly. Achieved dynamic visualization of fractal geometry, offering diverse exploration of self-similar patterns at varying scales.

Features: Additionally, optimized performance and enhanced user experience were ensured through efficient memory management and interactive zoom and pan functionalities, providing users with an immersive journey into the fascinating world of fractal geometry.

C MLX Library Fractals Mathematical Algorithms Computer Graphics

Minishell

I am currently collaborating on the Minishell project, replicating a bash terminal in C at 42. Implemented features include prompt display, history, executable search, re-directions, pipes, and built-ins. Utilizes readline function, avoids memory leaks, and adheres to project specifications. Duo project involving two team members.

Advanced Features: Additionally, Minishell incorporates advanced functionalities such as handling environment variables, quoting mechanisms, exit status retrieval, and signal handling. It ensures interactive mode responsiveness by appropriately handling ctrl-C, ctrl-D, and ctrl-\\ signals.

Quality Assurance: As part of the project, strict adherence to the norm guidelines and thorough testing practices are maintained to ensure code quality and reliability. With meticulous attention to detail, the Minishell project aims to provide a seamless and efficient user experience akin to a standard bash terminal.

C Shell Programming System Calls Process Management Signal Handling
